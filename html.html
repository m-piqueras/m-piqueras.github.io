About Me

I am a telecommunications engineer passionate about continuous learning and technological innovation. My experience focuses on areas such as sound and image, electronics, systems, and telematics, applying this knowledge to solve complex problems and develop practical and efficient solutions.

Throughout my academic career, I have participated in a wide variety of projects, including antenna design, FPGA programming, electronic boards, acoustics, signal coding and decoding, descriptor analysis applied to images, neural networks, speaker diarization, data encryption/decryption, and data traffic analysis. These initiatives have involved the use of tools and languages such as Python, MATLAB, Xilinx, Assembly Language, C, Vivado, Vitis, Simulink, Google Colab, Wireshark, ADS, CST, Linux terminal, and virtual machines, consolidating my ability to tackle multidisciplinary challenges and effectively coordinate and lead groups of people.

Key Projects

Bachelor's Thesis (TFG)

  • Designed a neural network architecture to predict turn-taking changes in audio recordings based on acoustic feature analysis.
  • Engineered a rigorous data preprocessing pipeline to tabularize and format acoustic inputs for optimal model training.
  • Developed a custom post-processing module to refine predictions and visualize results with high precision.

FPGA Audio System (HW/SW Co-Design)

  • Developed a real-time audio recording and playback system using Vivado and Simulink for hardware synthesis.
  • Implemented embedded control software in C using Vitis to manage audio codecs and data flow.
  • Integrated on-board peripherals to enable physical volume control and recording/playback mode switching directly from the FPGA board.

Network Planning & Traffic Analysis

  • Deployed network collectors and exporters to gather real-time telemetry and visualize traffic flow data.
  • Conducted deep packet inspection and flow analysis to identify bottlenecks, protocols, and usage patterns.
  • Dimensioned the network infrastructure and generated a cost-optimized, real-world budget proposal based on capacity requirements.

Biometric Signal Processing

  • Developed a biometric recognition system for online handwritten trajectories using Dynamic Time Warping (DTW) and Artificial Neural Networks (ANN).
  • Engineered a feature extraction pipeline selecting key kinematic descriptors (dX, dY, velocity, angle) and transforming temporal data into spatial heatmaps.
  • Designed and trained a Multi-Layer Perceptron (MLP) with ReLU activation and Dropout regularization, achieving 98.7% accuracy through rigorous cross-validation.

Embedded Systems & Microcontrollers

  • Developed a fully functional interactive memory game on a microcontroller-based embedded platform using C.
  • Designed a robust Finite State Machine (FSM) to manage complex game logic, including multiple game modes (Classic, Potentiometer, Dual), adaptive difficulty levels, and user input validation.
  • Integrated and controlled hardware peripherals including Joysticks, Potentiometers, Accelerometers (for motion detection), RGB LEDs, and LCD displays via low-level I/O interfaces.

Computer Vision & Image Processing

  • Developed an image search engine in MATLAB capable of retrieving similar images based on visual content (color and texture) rather than metadata.
  • Implemented and compared Color Layout Descriptors (CLD) using Discrete Cosine Transform (DCT) on YCrCb space vs. Raw Pixel matching.
  • Engineered a custom Texture Histogram Descriptor to improve classification accuracy, optimizing the system for a dataset of 280+ images.

RF & Microwave Engineering

  • Designed and simulated Microstrip Patch and Dipole antennas using CST Studio Suite, optimizing geometry for the 4.7 GHz frequency band.
  • Engineered the transmission line and impedance matching network using ADS (Advanced Design System) to ensure maximum power transfer.
  • Analyzed electromagnetic performance including Return Loss (S11), VSWR, and 3D Farfield radiation patterns to verify directivity and gain.
